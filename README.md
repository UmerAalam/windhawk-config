## 🚀 How to Use This Config

### 🔹 Step 1: Install Windhawk

1. Download from the official site
2. Run installer
3. Open Windhawk after installation

---

### 🔹 Step 2: Download This Repo

* Click **Code → Download ZIP**
* Extract it anywhere on your PC

---

### 🔹 Step 3: Restore My Config

1. Open **Windhawk Services Backup Utility**
2. Click **Restore**
3. Select the backup file from this repo
4. Confirm restore

🎉 That’s it — your setup will now match my configuration!

---

## 🔄 Important Notes

* 🔁 Restart Windhawk (or PC) after restoring
* ⚠️ Some mods may require manual enable
* 🧪 Works best on updated Windows 10/11
* 💡 If something breaks, just restore your old backup
